# JavaScript Greeting Page

This is a simple beginner JavaScript project that asks the user for their name and displays a greeting message.

## Features
- Enter your name
- Click button to generate greeting
- Dynamic JavaScript interaction

## Files Included
- index.html
- style.css
- script.js

## How to Run
Open `index.html` in your browser.
